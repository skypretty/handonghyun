Overview

여기어때 컴퍼니 : Visual System

여기어때의 비주얼은 꾸준히 고도화되고 있고, 초기 버전에 비해 훨씬 더 발전된 모습으로 확장되고 있어요. Simple, Scalable, Lively라는 코어 밸류를 기반으로 시작해 계속 발전 중인 비주얼 셋을 보여주고자 합니다.

Core Values

여기어때 Core Value

여기어때 비주얼 시스템이 지향하는 핵심 가치입니다. 모든 디자인 결정의 기준이 되는 원칙들로, 일관된 브랜드 경험을 위한 토대를 이룹니다.

Clarity

복잡함을 걷어내고 명확하게. 사용자가 즉시 이해할 수 있는 직관적인 시각 언어를 추구합니다.

Warmth

따뜻하고 친근한 감성. 여행과 여가의 설렘을 담아 사용자와 정서적으로 연결되는 디자인입니다.

Consistency

모든 터치포인트에서 하나의 목소리. 통일된 시각 언어로 브랜드 신뢰도를 높입니다.

Scalability

성장에 맞게 유연하게 확장되는 시스템. 새로운 서비스와 컨텍스트에도 일관성을 유지합니다.

2D Visuals & Guide

2D 그래픽 비주얼 시스템

여기어때 컴퍼니의 2D 그래픽 아이콘 시스템입니다. 음식, 여행, 레저 등 다양한 카테고리에 걸쳐 일관된 스타일로 제작된 2D 오브젝트들로, 서비스 전반의 비주얼 언어를 정의합니다.

Style Guide

플랫한 형태와 명확한 실루엣을 기반으로 제작된 2D 그래픽 스타일 가이드. 색상, 선 굵기, 여백 비율을 정의합니다.

Color System

카테고리별 색상 팔레트와 컬러 사용 원칙. 브랜드 컬러와의 조화를 고려한 오브젝트 색상 시스템입니다.

Grid & Sizing

100×100px 기준 그리드 시스템과 다양한 해상도 대응을 위한 사이즈 가이드라인입니다.

Usage Rules

UI 컴포넌트, 마케팅 소재, 서비스 화면 내 2D 오브젝트 활용 규칙과 금지 사항입니다.

3D Visuals & Guide

3D 그래픽 비주얼 시스템

2D 오브젝트와 동일한 아이덴티티를 유지하면서 입체감과 깊이를 더한 3D 그래픽 시스템입니다. 캠페인, 랜딩페이지, 고관여 콘텐츠에서 더 풍부한 시각적 임팩트를 제공합니다.

3D Style Guide

재질감, 조명 방향, 그림자 처리 등 3D 그래픽 전반에 적용되는 스타일 원칙을 정의합니다.

Light & Shadow

일관된 광원 방향과 그림자 표현 방식. 모든 3D 오브젝트에 통일된 조명 규칙을 적용합니다.

Material System

오브젝트 유형에 따른 재질 표현 가이드. 매트, 글로시, 반투명 소재별 렌더링 기준입니다.

Usage Rules

3D 비주얼이 활용되는 컨텍스트와 2D와의 혼용 가이드. 해상도 및 포맷 기준을 포함합니다.

Illustrations

일러스트레이션 시스템

서비스의 감성적 순간들을 위한 일러스트레이션 가이드입니다. 빈 상태, 에러 화면, 온보딩 등 사용자와 정서적으로 연결되는 지점에서 여기어때만의 따뜻한 시각 언어를 전달합니다.

Illustration Style

캐릭터 비율, 선 표현, 채색 방식 등 일러스트레이션 전반의 스타일 기준을 정의합니다.

Scene Composition

배경, 오브젝트, 캐릭터의 배치 원칙과 여백 처리. 다양한 화면 비율에 대응하는 컴포지션 가이드입니다.

Emotion Tone

각 상황별 감정 톤 가이드. 즐거움, 안심, 공감 등 서비스 맥락에 맞는 감성 표현 원칙입니다.

Application

Empty state, 에러 페이지, 온보딩, 이벤트 배너 등 실제 서비스 화면 내 일러스트 적용 사례입니다.

Design System Onboarding

디자인 시스템 온보딩

여기어때 비주얼 시스템을 처음 접하는 디자이너와 개발자를 위한 온보딩 가이드입니다. 시스템의 철학과 원칙부터 실제 적용 방법까지, 빠르게 이해하고 활용할 수 있도록 안내합니다.

Design Principles

여기어때 비주얼 시스템이 추구하는 핵심 가치와 디자인 철학. 모든 의사결정의 기준이 되는 원칙입니다.

Quick Start

Figma 라이브러리 연결부터 첫 번째 컴포넌트 사용까지. 온보딩 체크리스트로 빠르게 시작하세요.

Contribution Guide

시스템에 새로운 컴포넌트나 패턴을 기여하는 방법. 리뷰 프로세스와 버전 관리 규칙을 안내합니다.

Release Notes

비주얼 시스템의 변경 이력과 버전별 업데이트 내용. 주요 변경 사항과 마이그레이션 가이드를 포함합니다.
