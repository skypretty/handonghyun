Overview

여기어때 컴퍼니 : Visual System

여기어때의 비주얼은 꾸준히 고도화되고 있고, 초기 버전에 비해 훨씬 더 발전된 모습으로 확장되고 있어요. Simple, Scalable, Lively라는 코어 밸류를 기반으로 시작해 계속 발전 중인 비주얼 셋을 보여주고자 해요.

Core Values

여기어때 Core Value

여기어때 비주얼 시스템이 지향하는 핵심 가치예요. 모든 디자인 결정의 기준이 되는 원칙들로, 일관된 브랜드 경험을 위한 토대를 이뤄요.

Simple

불필요한 요소를 제거하고 본질에 집중해요. 사용자가 즉시 이해할 수 있는 직관적인 시각 언어를 추구해요.

Scalable

성장에 맞게 유연하게 확장되는 시스템. 새로운 서비스와 컨텍스트에도 일관된 시각 언어를 유지해요.

Lively

따뜻하고 생동감 있는 감성. 여행과 여가의 설렘을 담아 사용자와 정서적으로 연결되는 비주얼을 만들어요.

2D Visuals & Guide

2D 그래픽 비주얼 시스템

2D 그래픽 셋은 직관적이고 명확한 정보 전달을 목표로 하고 있어요. 여기어때만의 그래픽 아이덴티티를 반영하면서, 복잡한 텍스트나 설명 없이도 사용자가 정보를 쉽게 이해할 수 있도록 설계되어 있어요. 이를 통해 여기어때만의 차별화된 경험을 제공하고 있어요.

System Graphic

여기어때의 그래픽 시스템은 서비스 전반의 일관된 시각 언어를 형성해요. 플랫한 형태와 절제된 색상을 기반으로, 각 오브젝트는 브랜드 아이덴티티를 직관적으로 전달해요.

  • 100×100px 기준 그리드 시스템
  • 카테고리별 색상 팔레트와 컬러 원칙
  • 플랫한 형태와 명확한 실루엣
  • UI, 마케팅 소재 전반 활용 가능

2D 오브젝트

메타포를 활용할 때 가장 중요한 것은 전달하려는 기능, 목적, 또는 특징을 명확하게 강조하는 것이라고 생각했어요. 그래서 우리는 몇 가지 원칙을 기반으로 모든 그래픽들을 만들었어요.

Rule 01
그리드 시스템

모든 오브젝트는 100×100px 기준 그리드 위에 설계돼요. 일관된 비율과 여백을 유지해요.

Rule 02
컬러 팔레트

카테고리별로 정해진 색상 팔레트를 사용해요. 오브젝트당 최대 4색 이내로 제한해요.

Rule 03
선 굵기 원칙

외곽선은 2pt 기준으로 통일해요. 내부 디테일은 1pt로 처리하여 계층감을 부여해요.

Rule 04
실루엣 명확성

작은 크기에서도 식별 가능한 명확한 실루엣을 유지해요. 24px 이하에서도 인지 가능해야 해요.

Rule 05
라운드 처리

모서리와 끝점은 일관된 라운드 값을 적용해요. 날카로운 각보다 부드러운 형태를 지향해요.

Rule 06
배경 독립성

어떤 배경색 위에서도 오브젝트가 명확히 구분되어야 해요. 배경 의존적 표현은 지양해요.

3D 그래픽 시스템

2D 오브젝트와 동일한 아이덴티티를 바탕으로, 입체감과 깊이감을 더해 더욱 풍부한 시각적 임팩트를 전달해요. Blender 기반으로 제작된 고품질 3D 오브젝트들이에요.

  • 통일된 광원 방향과 그림자 원칙
  • 재질 유형별 렌더링 기준 (매트·글로시·반투명)
  • 2D와 동일한 카테고리 체계 적용
  • 캠페인·랜딩페이지·고관여 콘텐츠 활용

3D 오브젝트

음식, 여행, 레저 카테고리 전반에 걸쳐 제작된 3D 오브젝트예요. 2D 그래픽과 동일한 실루엣을 유지하면서 재질감과 조명으로 입체감을 부여해요.

서비스 내 특별한 순간과 고관여 컨텍스트에서 사용자 경험에 깊이와 감성을 더하는 시각 자산으로 활용돼요.

Rule 01
광원 방향

모든 3D 오브젝트에 통일된 광원 방향을 적용해요. 상단 좌측 45° 각도를 기준으로 일관된 조명을 구현해요.

Rule 02
재질 표현

오브젝트 특성에 따라 매트, 글로시, 반투명 세 가지 재질 타입 중 적합한 유형을 선택하여 적용해요.

Rule 03
그림자 처리

그림자는 오브젝트 하단에 부드럽게 드리워지도록 처리해요. 과도한 그림자 표현은 지양해요.

Rule 04
형태 일관성

2D 오브젝트의 실루엣을 기반으로 3D 형태를 제작해요. 동일한 카테고리 체계와 비율 원칙을 유지해요.

Rule 05
색상 시스템

2D와 동일한 색상 팔레트를 기반으로 해요. 재질감 표현을 위한 명도·채도 조정은 허용 범위 내에서 가능해요.

Rule 06
렌더링 기준

최종 산출물은 PNG 투명 배경으로 내보내요. 해상도는 최소 2x 이상을 원칙으로 하며 포맷별 기준을 따라요.

Illustrations

일러스트레이션 시스템

서비스의 감성적 순간들을 위한 일러스트레이션 가이드예요. 빈 상태, 에러 화면, 온보딩 등 사용자와 정서적으로 연결되는 지점에서 여기어때만의 따뜻한 시각 언어를 전달해요.

Illustration Style

캐릭터 비율, 선 표현, 채색 방식 등 일러스트레이션 전반의 스타일 기준을 정의해요.

Scene Composition

배경, 오브젝트, 캐릭터의 배치 원칙과 여백 처리. 다양한 화면 비율에 대응하는 컴포지션 가이드예요.

Emotion Tone

각 상황별 감정 톤 가이드. 즐거움, 안심, 공감 등 서비스 맥락에 맞는 감성 표현 원칙이에요.

Application

Empty state, 에러 페이지, 온보딩, 이벤트 배너 등 실제 서비스 화면 내 일러스트 적용 사례예요.

Design System Onboarding

디자인 시스템 온보딩

여기어때 비주얼 시스템을 처음 접하는 디자이너와 개발자를 위한 온보딩 가이드예요. 시스템의 철학과 원칙부터 실제 적용 방법까지, 빠르게 이해하고 활용할 수 있도록 안내해요.

Design Principles

여기어때 비주얼 시스템이 추구하는 핵심 가치와 디자인 철학. 모든 의사결정의 기준이 되는 원칙이에요.

Quick Start

Figma 라이브러리 연결부터 첫 번째 컴포넌트 사용까지. 온보딩 체크리스트로 빠르게 시작하세요.

Contribution Guide

시스템에 새로운 컴포넌트나 패턴을 기여하는 방법. 리뷰 프로세스와 버전 관리 규칙을 안내해요.

Release Notes

비주얼 시스템의 변경 이력과 버전별 업데이트 내용. 주요 변경 사항과 마이그레이션 가이드를 포함해요.
