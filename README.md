## About

이 포트폴리오는 결과물을 나열하는 공간이 아니라  
브랜드 맥락과 UX 구조를 기반으로 한 설계 과정을 담고 있습니다.

브랜드는 시각 요소의 조합이 아니라  
일관된 기준 위에서 작동하는 시스템이라고 생각합니다.

UX는 화면의 형태보다  
정보 위계와 사용자 흐름의 논리에서 출발합니다.

프로덕트는 기능이 아니라  
연속된 경험의 설계라고 보고 있습니다.
